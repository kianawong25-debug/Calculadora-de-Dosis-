# Calculadora de Dosis Médicas

Aplicación desarrollada con Flutter para calcular dosis de medicamentos en pacientes pediátricos y adultos.

## Funciones

- Cálculo de dosis por peso (mg/kg)
- Cálculo de dosis para adultos
- Conversión a mL para jarabes
- Estimación de frascos necesarios para el tratamiento

## Tecnologías

- Flutter
- Dart
